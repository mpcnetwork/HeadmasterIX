Cursus MySQL

Introductie

In deze cursus leer je hoe je gegevens in een database kunt aanmaken, bewerken, opvragen en verwijderen. Dit doen we met behulp van MySQL, één van de populairste relationele databasesystemen ter wereld.

Wat is SQL?

SQL staat voor Structured Query Language. Het is de standaardtaal om relationele databases te beheren. Met SQL kun je tabellen aanmaken, relaties leggen en data manipuleren.

Wat is MySQL?

MySQL is een implementatie van SQL. Het ondersteunt de standaard SQL-commando's, maar voegt ook eigen uitbreidingen toe. Andere implementaties zijn Oracle, PostgreSQL en Microsoft SQL Server.

Hoe leer je in deze cursus?

  • Belangrijke begrippen en commando's worden stap voor stap uitgelegd.
  • Bij elk onderdeel vind je voorbeelden en oefeningen.
  • Elke module eindigt met opdrachten om je kennis te toetsen.

Database

Een database is een georganiseerde verzameling gegevens die op een gestructureerde manier wordt opgeslagen en beheerd door een computersysteem.

Relationeel versus niet-relationeel

  • Relationele databases (RDBMS): Gegevens in tabellen met sleutels. Voorbeelden: MySQL, PostgreSQL, Oracle, SQL Server.
  • Niet-relationele databases (NoSQL): Flexibele structuur voor ongestructureerde data. Voorbeelden: MongoDB, CouchDB, Redis.

In deze cursus focussen we op relationele databases, met MySQL als praktische omgeving.

Waarom relationele databases?

  • Snel terugvinden van informatie dankzij indexing en query-optimalisatie.
  • Relaties tussen tabellen via sleutels.
  • Efficiënt opslaan van data zonder duplicatie.
  • Beveiliging en rechtenstructuur voor verschillende gebruikers.
  • Gestandaardiseerde taal (SQL) die werkt in vrijwel alle systemen.

Installatie

Om lokaal te oefenen met MySQL en phpMyAdmin, gebruiken we een webserverpakket zoals WampServer.

Stap 1: Kies een serverpakket

  • WampServer (Windows)
  • XAMPP (Windows, macOS, Linux)
  • MAMP (macOS, Windows)

In deze cursus gebruiken we WampServer als voorbeeld.

Stap 2: Installatie

  1. Download de juiste versie van WampServer (meestal 64-bit).
  2. Voer het installatiebestand uit en volg de instructies.
  3. Controleer het icoontje in de taakbalk:
    • Rood: server uitgeschakeld
    • Oranje: gedeeltelijk actief
    • Groen: alles correct gestart

Stap 3: phpMyAdmin openen

  1. Klik op het groene WampServer-icoon.
  2. Kies phpMyAdmin in het menu.
  3. Log in met:
    • Gebruikersnaam: root
    • Wachtwoord: leeg

Let op: Dit is enkel veilig in een lokale omgeving. Gebruik een sterk wachtwoord op een echte server.

Stap 4: Eerste kennismaking met phpMyAdmin

  • Nieuwe databases aanmaken

  • Gebruikersrechten beheren

Onderaan het startscherm zie je info over:

  • Webserver: Apache

Storage engines in MySQL

  • InnoDB: Standaard sinds MySQL 5.5. Ondersteunt relaties en transacties. Aanbevolen keuze.
  • MyISAM: Verouderd. Geen transacties of foreign keys. Niet meer aanbevolen.

Zorg dat je tabellen met InnoDB aanmaakt. Dit is meestal standaard actief.

Downloadlink WampServer

Surf naar wampserver.aviatechno.net en download de juiste versie (meestal x64).

Installatieproces

  1. Dubbelklik op het uitvoeringsbestand in je downloadsmap.
  2. Kies de taal van de installatie (bijv. Engels).
  3. Accepteer de licentievoorwaarden (GNU LGPL).
  4. Lees de installatie-informatie en klik op Next.
  5. Controleer of je systeem de juiste Visual C++ redistributables heeft (VC9 t/m VS16).
  6. Installeer WampServer niet over een bestaande versie.
  7. Kies een installatiemap zonder spaties of speciale tekens (bijv. C:\wamp).
  8. Sluit programma’s zoals Skype die poort 80 gebruiken.

Tips voor eerste gebruik

1. Start WampServer correct op

  • Dubbelklik op het WampServer-icoon op je bureaublad.
  • Wacht tot het icoon in de taakbalk groen wordt (alle services actief).
  • Indien het icoon oranje blijft, controleer of Apache of MySQL niet geblokkeerd worden door andere programma’s (zoals Skype).

2. Open phpMyAdmin

  • Klik met de linkermuisknop op het groene WampServer-icoon.
  • Kies phpMyAdmin in het menu.
  • Log in met gebruikersnaam root en laat het wachtwoordveld leeg.

3. Maak je eerste database aan

  1. Ga in phpMyAdmin naar het tabblad Databases.
  2. Geef een naam op (bijv. oefendb) en kies Collation als utf8_general_ci.
  3. Klik op Aanmaken.

4. Maak een eerste tabel

  1. Open je nieuwe database.
  2. Geef een tabelnaam op (bijv. klanten) en aantal kolommen.
  3. Definieer kolommen zoals id, naam, email.
  4. Stel id in als PRIMARY KEY en AUTO_INCREMENT.

5. Voer je eerste SQL-query uit

Ga naar het tabblad SQL en probeer bijvoorbeeld: 

INSERT INTO klanten (naam, email) VALUES ('Friedel', 'friedel@example.com');

6. Veiligheid en best practices

  • Gebruik sterke wachtwoorden zodra je buiten een lokale omgeving werkt.
  • Maak regelmatig backups van je databases.
  • Experimenteer eerst in een testdatabase voordat je live-data aanpast.
  • Leer het verschil tussen SELECT, INSERT, UPDATE en DELETE.
